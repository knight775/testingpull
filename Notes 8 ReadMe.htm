Notes 8.5.x - Client Removal Package - Where did my preferences go?

 

Question:

 

I ran the default Notes install bundle from ISSI and had to run the "Lotus Notes Client Removal Tool V1.xx". What happened to my preferences?

 

Answer:

 

Your Lotus Notes data, bookmarks, desktop, location documents and Sametime contacts are NOT removed by the Removal Tool. A backup copy of Sametime Emoticons and Primary contacts are saved to C:\Notes_Preferences_Backup and can be imported back into Notes embedded Sametime. 

Preferences stored in the Notes File/Preferences panels, and other integration features, such as custom Widgets and Plug-ins are removed. 

 

Please visit the following link in IT Help Central for the Lotus Notes Post Installation Checklist.

 

Notes 8 Standard or Basic - What does this mean to me?

 

Question:

 

I understand that the Notes 8 client can be run in one of two ways - Standard or Basic configuration. What does this mean to me?

 

 

Answer:

 

When you install Lotus Notes 8 you will have two forms of the client available to you.

 

The Basic Notes configuration, as the name implies, is the traditional, Notes only, version of the Notes client. Looking and working much like previous Notes releases, this configuration includes only a subset of the new features available in Notes 8, but does use less memory. This makes it an option if you experience performance issues when running the full Standard configuration. The major difference between this configuration and the Standard form is that the Basic configuration does not allow any plug in capability. This means that Lotus Symphony and Lotus Sametime must be used outside the Notes client, which may offset some of the memory gains.

 

The Standard configuration is based on Eclipse technology, and offers a much higher level of functionality than the Basic configuration. The Eclipse framework offers the opportunity to extend the capabilities of Notes with additional 'plug in' applications, including Lotus Symphony, Lotus Sametime 8.0, and Activities, all of which are part of the installation package from ISSI. Any of these plug in applications can be upgraded separately from the Notes client, removing the need to install Notes multiple times. This also allows you to further extend the function of Notes, Symphony, and Sametime with additional plug ins which you can install yourself.

 

For a more detailed description of the functional differences between the two configurations, see the Help Central document titled Notes 8 Standard or Basic - What are the functional differences? 

 

 

Why should I choose one configuration over the other?

 

Under ideal circumstances, the Standard configuration should be your first choice. It offers a much greater variety of additional function, and allows you to run Notes, Lotus Symphony, and Lotus Sametime Connect, all in the same Eclipse framework, enhancing productivity and potentially improving workstation performance by allowing several major software components to share resources.

 

The major reasons for choosing the Basic configuration are related to workstation and client performance. Testing has shown that the Standard configuration performs well under most IBM workstation profiles, but individual requirements can affect that situation. If you find that performance is less that you require when running the Standard configuration, switching to the Basic configuration may afford you a significant performance increase. Note that in this configuration, running additional software such as Lotus Symphony and Lotus Sametime Connect in stand alone mode can use more memory than the embedded versions. This could reduce the performance impact.

 

 

How do I get the Basic configuration?

 

Anybody that installs Notes 8 from the ISSI will automatically have the ability to run Notes in its Basic configuration. The install package adds desktop icons for starting both the Basic and Standard configurations. This allows you to choose which configuration to start, depending on your specific circumstances. The Standard configuration will, however, be set as the default, unless you have less than 1 Gb of memory on your workstation, in which case the Basic configuration is set as the default. See the section on setting the default configuration for what this means to you.

 

 

Can I use both Basic and Standard configurations?

 

You will not be able to run both configurations at the same time, but since both configurations use the same local data and location settings, there is no reason why you cannot switch between them as appropriate. For example, during times of minimum workstation usage, you could be running the Standard configuration with the integrated Sametime and Symphony clients available. When workstation usage becomes more intensive - you might be running development or knowledge mining tasks for instance - then you could switch to the Basic configuration and the stand alone Sametime client.

 

To switch from one configuration to the other, shut down Lotus Notes, then double click the icon for the alternative configuration. While the interface is different, data and location settings are shared between the two configurations, so you can continue to work with whatever you require.

 

 

Setting the default configuration

 

The installation will define the Standard configuration as default for your workstation. What this means to you is that when Notes is started by any action outside of Notes itself, it will be the Standard configuration that is started. Such actions include: double clicking on a Notes database from Windows Explorer, clicking a link to a Notes database in a Sametime chat, or opening a 'mail to' URL in a web page.

 

This default is controlled by a setting in your notes.ini file. Should you prefer that the Basic configuration start by default, you can change the setting as follows.

1.    Locate the notes.ini file (usually in c:\notes)

2.    Open the file for editing (Wordpad, or any other text editor)

3.    Locate the line item UseBasicNotes=0

4.    Change the zero to one (UseBasicNotes=1)

5.    If the line does not exist, you may add it in the last position

6.    Save your changes

 

 

I seem to have a lot of new desktop icons after the install. What are they all?

 

The Notes 8 installation package on ISSI, in common with previous install packages, offers the options of a 'client only' install, or the installation of additional Domino Designer and Domino Administrator clients. If you choose the latter installation you will see all of the following icons added to your desktop, and to your Windows start menu.

 

1.    Lotus Notes 8

2.    Lotus Notes 8 (Basic)

3.    Lotus Domino Designer 8

4.    Lotus Domino Administrator 8

5.    Lotus Symphony Documents

6.    Lotus Symphony Presentations

7.    Lotus Symphony Spreadsheets

8.     Lotus Notes Read Me (this icon opens this document)

 

In addition, if you have installed the stand-alone Lotus Symphony client, you will have a desktop icon for that also. All of these icons have a consistent colour scheme, and may look similar at first glance. If you find this confusing, and plan to use Lotus Symphony from within Lotus Notes, you can remove unwanted shortcuts by right clicking on the icon, and selecting Remove. You may also rename the icons in the same way if you feel this would help.
